## lombok

logback 是 java 的一个开源的日志框架,它提供了两个日志框架的日志，一个是 Log4j 一个是 Logback。
jd-gui 反编译，查看生成后的源码。

### ide 安装

- idea 安装 lombok 插件
- eclipse
  - 命令行:java -jar lombok.jar
  - 选择 eclipse 安装目录

### 项目引入 lombok 依赖

```
<dependency>
    <groupId>org.projectlombok</groupId>
    <artifactId>lombok</artifactId>
    <version></version>
    <scope>provided</scope>
</dependency>
```
