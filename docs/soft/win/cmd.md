# win 常见 cmd 命令

> `win+r 输入cmd` 进入命令行界面，需要权限时`以管理员身份运行命令提示符`.

> win 下引用系统变量 ，示例：`%JAVA_HOME% %M2_HOME%`

> 在命令行下运行软件需要把可执行程序所在的目录添加到系统的`Path`变量中

#### 常用命令

1. 进入某个盘

   ```
   //进入d盘
   D:
   //进入F盘
   F:
   // 进入文件夹
   cd
   ```

2. 查看目录

   ```
    dir
   ```

3. 查看 ip

   ```
   ipconfig
   ```

4. 清屏

   ```
   cls
   ```

5. 测试网络是否畅通

   ```
    ping ip
   ```

6. 修改查看注册表

   ```
    regedit
   ```

7. 查看系统服务

   ```
    services.msc
   ```

8. 删除系统服务

   ```
    // 需要管理员权限
    sc delete 服务名
   ```
